# toys

## 🎮 みんなのゲームランド (kids-game)

年齢別に難易度・あそびかたが変わる子供向けWebゲームです。

### あそびかた
`kids-game/index.html` をブラウザで開くだけ。または:

```sh
cd kids-game
python3 -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

### 年齢別モード
| 年齢 | あそび |
|------|--------|
| 3〜5さい 🧸  | いろ・どうぶつ えらび（絵文字＋音声よみあげ） |
| 6〜8さい ✏️  | たしざん・ひきざん（全10問） |
| 9〜12さい 🚀 | かけざん 30秒タイムアタック |

正解・不正解で音とアニメーション、最後に結果画面が出ます。
画像アセット不要・依存ライブラリ無しのバニラJSで動きます。
