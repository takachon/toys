# toys 🎮 みんなのキッズゲーム

年齢別に、子供が遊べる簡単なブラウザゲーム集です。
ライブラリ不要のバニラ HTML / CSS / JavaScript で作られています。

## あそびかた

`index.html` をブラウザで開くだけで遊べます。
ローカルサーバーで開く場合：

```bash
python3 -m http.server 8000
# → ブラウザで http://localhost:8000 を開く
```

1. 「としを えらんでね」で年齢を選ぶ
2. 遊びたいゲームを選ぶ
3. 遊ぶ！ 正解すると ⭐ スコアがたまります
4. 「🏠 ホーム」でいつでも最初に戻れます

## 年齢別ゲーム

| 年齢 | ゲーム | 内容 |
|------|--------|------|
| 3〜4さい 🍼 | どうぶつタップ | 出てくる動物をタップ（反射・指あそび） |
| 3〜4さい 🍼 | いろあわせ | 言われた色を選ぶ（色の認識） |
| 5〜6さい 🧸 | かずをかぞえよう | 絵の数を数える（数の認識） |
| 5〜6さい 🧸 | かたちあわせ | 言われた形を選ぶ（図形） |
| 7〜9さい 🎒 | けいさんチャレンジ | たし算・ひき算クイズ |
| 7〜9さい 🎒 | きおくゲーム | 神経衰弱（記憶力） |
| 10さい〜 🚀 | もぐらたたき | 30秒で何匹たたけるか（反射神経） |
| 10さい〜 🚀 | かけざんスピード | かけ算クイズ |

スマホ・タブレットのタッチ操作にも対応しています。

## テスト

jsdom で各ゲームを実際に起動・操作して動作を検証します。

```bash
npm install
npm test
```

## ファイル構成

- `index.html` — 画面構造
- `style.css` — 子供向けのカラフルなデザイン
- `game.js` — ゲームロジック（年齢グループ・全ミニゲーム）
- `test.js` — 動作検証テスト
