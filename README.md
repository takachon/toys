# toys 🎮 みんなのキッズゲーム

年齢別に、子供が遊べる簡単なブラウザゲーム集です。
ライブラリ不要のバニラ HTML / CSS / JavaScript で作られています。

## あそびかた

`index.html` をブラウザで開くだけで遊べます。
ローカルサーバーで開く場合：

```bash
python3 -m http.server 8000
# → ブラウザで http://localhost:8000 を開く
```

1. ホームで遊びたい枠（**ねんれいべつ** / **トランプゲーム**）を選ぶ
2. ゲームを選んで遊ぶ！ 正解すると ⭐ スコアがたまります
3. 「🏠 ホーム」でいつでも最初に戻れます

## トランプゲーム（1人でCPUと対戦）

年齢別とは別の枠。標準の52枚デッキを使い、すべて1人でCPU相手に遊べます。

| ゲーム | 人数 | 内容 |
|--------|------|------|
| しんけいすいじゃく 🃏 | あなた vs CPU | 16枚から同じカードのペアを多く取った方が勝ち。CPUはめくられた札を記憶する |
| 7ならべ 7️⃣ | あなた vs CPU3人 | 7から数字をつなげて手札を最初になくした人が勝ち。置けないときはパス |
| ババぬき 🤡 | あなた vs CPU | ペアを捨てつつ引き合い、最後にジョーカーを持っていたら負け |

## テスト

jsdom で各ゲームを実際に起動・操作して動作を検証します。

```bash
npm install
npm test
```

## ファイル構成

- `index.html` — 画面構造
- `style.css` — 子供向けのカラフルなデザイン
- `game.js` — ゲームロジック（カテゴリ・年齢グループ・全ミニゲーム・トランプゲーム）
- `test.js` — jsdom による動作検証テスト
- `screenshot.js` — Playwright で実ブラウザのスクリーンショットを撮る検証（`npm run screenshot`）
